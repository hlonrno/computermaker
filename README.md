# Title

This is a WIP kinda-clone of [CM2](https://www.roblox.com/games/6652606416/Circuit-Maker-2) in C. \
The original is made in Roblox which limits the speed of the simulation enough to be exceeded. \
We have no idea of computer graphics.

## Quickstart (Linux)

-# a nice [shell.nix](https://github.com/Qxtdl/computermaker/blob/master/shell.nix) has been provided for nixos users.
Requirements:
- [GLFW](https://www.glfw.org)
- [cglm](https://github.com/recp/cglm)

To run the app:
```sh
./build.sh
```

Your IDE is complaning about includes? \
Try `bear -- make`, that should fix it.
